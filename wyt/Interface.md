# IF 段

### 输入部分
|  变量名   |  位宽  |            功能             |  来自  |
| :-------: | :----: | :-------------------------: | :----: |
|    clk    |   1    |          全局时钟           |  全局  |
|    rst    |   1    |          全局复位           |  全局  |
|   Jump    |   1    |        Jump 是否成功        |   ID   |

|  EPC_sel  |   1    |     选择 PC 是否为 EPC      |   ID   |
|    EPC    | [31:0] |       作为 NPC 的 EPC       |   ID   |
